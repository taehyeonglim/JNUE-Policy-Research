## 최종 권고: 방안 C "야간+주말 혼합 모델"

| 설계 요소 | 내용 |
|-----------|------|
| 시간대 | 평일 야간 2회(18:00~21:00) + 격주 토요일 1회(09:00~12:00) |
| 원격 비율 | 수업일수의 2/3 이내 → 대면 출강 학기당 5~6회 |
| 겸임 교원 시수 | 학부 6시간 + 대학원 3시간 = 9시간 (규정 제25조 유연조항) |
| 석박 공통과목 | 연구방법론 등 최대 6학점 이내 (42학점 중 14%) |
| 선례 | 대구교대 "주/야간+주말" 모델과 동일 구조 |

## 핵심 발견

1. **반박가 논거2(전공일치) 무력화** — 제22조의1은 학부 교과교육과목 규정이지 대학원 강의 배정 규정이 아님. 대학원은 별도 학칙(제4조)으로 운영
2. **원격 2/3이 핵심 완충장치** — 대면 출강 월 1~2회로 축소되면 "매주 야간 근무" 전제 자체가 부정
3. **유연학기제 활용** — 대학원 집중 학기(6+3=9) / 비집중 학기(9+0=9) 교대 운영 가능
