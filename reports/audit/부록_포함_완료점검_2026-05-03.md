# 부록 포함 완료 점검

점검일: 2026-05-03

## 목표

최종보고서에 독자의 이해와 신청서 전환을 돕는 부록을 충실히 포함한다.

## 완료 기준과 증거

| 요구사항 | 반영 위치 | 증거 |
|---|---|---|
| 부록 1 A1 교원 의향조사 설문지 | `reports/chapters/10_참고문헌_부록.md` | 조사 개요, 기본 정보, A1-5~A1-21 핵심 문항, 판정 기준 포함 |
| 부록 2 A2 전북 현직교사 수요조사 설문지 | 같은 파일 | 표본 배분, A2-1~A2-17 핵심 문항, 5개 조건 유효수요 산식, 결과 보고표 포함 |
| 부록 3 교원 7명 후보 검증표 | 같은 파일 | 후보 T-01~T-07 검증표, 연구실적 환산표, 종합 판정 기준 포함 |
| 부록 4 재정 추계 산식표 | 같은 파일 | 변수 정의, 14개 충원 시나리오, 대표 6개 민감도 표, 필수 보강 자료 포함 |
| 부록 5 학칙·규정 개정 패키지 | 같은 파일 | 5-A부터 5-I까지 학칙, 교육전문대학원 학칙, 학사운영, 직제, 보수, 인사, 업적평가, 학위수여, 교연비 지급 기준 포함 |
| 부록 6 원자료 추적표 | 같은 파일 | 핵심 주장별 원자료·본문 위치·감사 ID 매핑, 감사 산출물, 남은 확인 필요 항목 포함 |
| 최종 HWPX 포함 | `reports/전주교대_교육전문대학원_설치_정책연구_초안.hwpx` | HWPX 내부에서 `부록 1.`~`부록 6.`, `A1-21`, `A2-17`, `14개 충원 시나리오`, `부록 5-I`, `핵심 추적표` 확인 |
| Pages 포함 | `docs/index.html` 및 `docs/assets/...hwpx` | Pages 본문과 다운로드용 HWPX 모두 재생성 |

## 검증 결과

- `python3 scripts/generate_hwpx.py`: 10개 장을 읽고 총 1,602개 블록으로 HWPX 재생성.
- `python3 scripts/generate_pages_site.py`: `docs/index.html` 재생성.
- `python3 scripts/audit_report_facts.py`: 본문 정합성 검사에서 학교별 새 CHECK 0건, 집계 새 CHECK 0건.
- 최종 HWPX 크기: 89,562 bytes. 부록 확장 전 84,660 bytes에서 증가.

## 남은 전제

부록 양식은 보고서에 포함되었지만, A1·A2 실제 조사 결과, 교원 실명 인사자료, 등록금·운영비 원자료는 아직 존재하지 않는다. 이 항목은 신청서 제출 전 원자료 확보 후 본 부록 양식에 입력해야 한다.
