# 문서형 원본 10건 MD 변환 완료 검증

검증일: 2026-05-03

## 1. 목적

이전 검토에서 직접 `.md` 또는 `_raw_text/*.txt` 변환물이 없다고 확인된 HWP 5건, HWPX 5건을 모두 Markdown으로 변환하고, 원본 추출 텍스트가 `.md`에 누락 없이 보존되는지 검증했다.

## 2. 변환 방식

| 원본 유형 | 변환 방식 | 산출물 |
|---|---|---|
| HWP | `hwp5txt`로 원문 텍스트 추출 후 Markdown 구조화 | 같은 폴더의 `*.md`, `_raw_text/*.txt` |
| HWPX | `Contents/section*.xml`의 `hp:t` 텍스트와 표 셀을 직접 추출 후 Markdown 구조화 | 같은 폴더의 `*.md`, `_raw_text/*.txt` |

변환 스크립트는 `scripts/convert_unconverted_docs.py`에 추가했다.
