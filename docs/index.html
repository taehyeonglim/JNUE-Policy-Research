전주교대 위치

전주교대는 호남권 초등교육 전문성 고도화, 박사급 현장 연구자 양성, 교육대학원 구조 개선이라는 세 과제를 동시에 안고 있습니다. 교육전문대학원 설치는 이 세 문제를 하나의 제도 설계로 묶는 방식입니다.

2. 전공 비교

4. 설치 당위성

정책 흐름

2025년 5개 교대가 병설형 교육전문대학원을 설치하면서, 교육대학교 박사과정 운영은 예외적 모델이 아니라 확산 단계로 이동했습니다.

지역 책무

전북권 초등교육 연구·연수·정책 실행 역량을 지역 내부에서 축적하려면, 석사 재교육 중심 대학원만으로는 한계가 있습니다.

운영 원칙

1기 정원은 20~24명, 실제 개설 전공은 4~5개부터 시작하고, 충원율과 지도교수 확보를 확인한 뒤 단계적으로 확대하는 보수적 모델이 적절합니다.

교원 확보

핵심 리스크는 전임교원 7명 소속 이동입니다. 학부 6시간 + 대학원 3시간의 겸보 모델, 책임시수 통합 산정, 교연비·업적평가 실적 인정이 함께 설계되어야 합니다.

5. 학칙 개정 핵심

전주교대 학칙 개정은 단순히 제3조에 교육전문대학원을 추가하는 방식으로는 부족합니다. 전임교원 7명의 신분보장, 학부 수업권 보호, 책임시수 인정, 교연비·업적평가 불이익 금지를 함께 담는 신분보장형 패키지 개정이 필요합니다.

본칙 핵심 조항

제4조의2를 신설해 전임교원의 소속 변경, 겸무·겸보, 학부 겸무 범위에서의 소속 간주, 학부·교육대학원·교육전문대학원 강의시간 통합 산정, 불이익 금지 원칙을 둡니다.

하위규정 연동

교연비 금액이나 배점을 학칙에 직접 쓰지 않고, 학칙은 실적 인정과 불이익 금지 원칙을 둡니다. 세부 기준은 학사운영규정, 교원업적평가규정, 교육·연구 및 학생지도 비용 지급 기준으로 위임합니다.

6. 배포 구조

이 페이지는 서버 프로세스가 필요 없는 정적 HTML입니다. 저장소에는 자동 생성 스크립트 scripts/generate_pages_site.py, GitHub Actions 워크플로 .github/workflows/pages.yml, 정적 산출물 docs/index.html이 함께 있습니다.

GitHub Pages 설정에서 배포 소스를 Deploy from a branch로 둘 경우 main / docs를 선택하면 이 파일이 바로 공개됩니다. GitHub Actions 방식을 쓰는 경우에는 저장소 Pages 기능을 먼저 활성화한 뒤 워크플로가 생성한 산출물이 배포됩니다.
