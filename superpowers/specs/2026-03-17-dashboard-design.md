# 교육전문대학원 현황 비교 대시보드 설계

## 목적
전주교육대학교 교육전문대학원(박사과정) 설치의 당위성을 입증하기 위해, 타 교육대학교 교육전문대학원 현황을 비교 분석하는 웹 대시보드를 구축한다.

## 기술 스택
- **Python 3.11+**
- **Streamlit** — 대시보드 프레임워크
- **Pandas** — 데이터 전처리
- **Plotly** — 인터랙티브 차트
- 향후: python-docx (보고서 생성)

## 프로젝트 구조

```
JNUE-Policy-Research/

│   └── pages/
│       ├── 01_현황개요.py
│       ├── 02_전공비교.py
│       ├── 03_학생통계.py

```

## 대시보드 페이지 구성

### 페이지 1: 현황 개요
- 10개 교대 교육전문대학원 설치 현황 테이블
- 설치 연도 타임라인 차트
- 핵심 지표 카드 (설치 비율, 미설치 대학 수)

### 페이지 2: 전공 비교
- 대학별 전공 목록 테이블
- 전공 분야별 히트맵 (공통 전공 파악)

### 페이지 3: 학생 통계
- 대학별 대학원 재학생 수 비교 (막대 차트)
- 연도별 추이 (선 그래프)
- 석사/박사 비율

### 페이지 4: 경쟁률 분석
- 대학별 신입생 경쟁률 비교
- 연도별 경쟁률 추이

## 데이터 흐름
1. KESS 엑셀 → data_loader.py (pandas 파싱)
2. analyzer.py (필터링, 집계, 비교)
3. charts.py (plotly 차트 생성)
4. Streamlit 페이지에서 렌더링

## 초기 전략
- 조사된 데이터 기반 샘플 데이터로 프로토타입 구동
- KESS 데이터 투입 시 data_loader.py만 연결하면 자동 갱신
